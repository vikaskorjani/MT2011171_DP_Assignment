Steps to execute the assignment :

1. Import the project org.iiitb.MT2011171.PlanMyTripApplication in Eclipse.
2. import  "tripdetails.sql" database in mysql.
   use following command: 
	- go to the location of bin in cmd
	- create database in mysql : tripDetailsData
	- run the command : mysql -u root -p tripDetailsData < tripPlanner.sql
4. In Project org.iiitb.MT2011171.PlanMyTripApplication go to /utils/RunTimeSetting.java file and give the password of Your mysql server.
5. Run the project as eclipse Application.
